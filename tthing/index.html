how to add my repo to package manager

open package manager

hover "tools"

press edit sources

and your source file should look like this

{
    "sources": [
        "https://packages.windows96.net/main",
        "https://woozier.github.io/tthing"
    ],
    "updateFrequency": 2

and close package manager then open it again

now you have my repo in the package manager
